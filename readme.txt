These files contain the CMLS beamer theme:
beamercolorthemeCMLS.sty - color setup
beamerouterthemeCMLS.sty - outer theme setup
beamerthemeCMLS.sty - general setup
images/frontpage-bgp.pdf - background inmage for titlepage
images/slides-bgp.pdf - background inmage for slides
template.tex - an example this theme


The workflow:

Put all three '*.sty' files ang 'images' folder in the same directory as
'template.tex'. Unzipping this archive produces a correct file structure.

Compile 'template.tex' using PDFLaTeX or XeLaTeX.
